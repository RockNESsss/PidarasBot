# Telegram Рейтинг Бот

Цей бот дозволяє створювати "кола", додавати учасників, змінювати рейтинг, призначати редакторів та щотижнево визначати "підараса дня" або видавати жартівливі титули.

## Установка

1. Встав свій токен у config.py
2. Встанови залежності:
    pip install -r requirements.txt
3. Запусти файл bot.py
